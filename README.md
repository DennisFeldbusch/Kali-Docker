# About
This repository should be used to dockerize a Kali Linux environment.

# Requirements

1. `docker --version`
2. `chmod +x start.sh`
3. `chmod +x connect.sh`
4. move ovpn file in current directory

# Usage

`./start.sh <filename of ovpn file WITHOUT extension>`

## Example

`./start.sh htb`
